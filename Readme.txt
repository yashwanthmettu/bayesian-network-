Structure of the code:

A single python file is developed named bnet.py. In this code, the probabilities in the tables are utilized and developed a Bayesian network among all of them. The code prints the probability of the query that is given by the user


Libraries used:
sys: Used this library to interact the python file with the command line arguments, sys.argv provides the access for it.

Need to run the code from terminal (command_line_argument)


the order to follow to run the code is--> bnet query

Example to run the code:

--> python bnet.py Jt given Et Bf

--> python bnet.py Jf Mt given Et

--> python bnet.py Jf Mf given Ef
